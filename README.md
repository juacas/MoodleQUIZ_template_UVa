## English (en)

### 1. Purpose

Create and export quiz questions from Word to Moodle (GIFT format) efficiently.

### 2. Features

- Ribbon buttons for each question type.
- Automatic template insertion.
- Validation helpers.
- Multi-language interface.
- Weighted answers support.

### 3. Installation

1. Download the template file (Word_Moodle_quiz.dotm).
2. If macros are blocked: Right‑click the .dotm > Properties > check “Unblock” > OK.
3. Double‑click the .dotm to open a new document based on the template.
4. Click “Enable Content” to allow macros.
5. Use the “Exportar a Moodle” / “Export to Moodle” ribbon tab buttons.

### 4. Basic Workflow

1. Insert question template.
2. Fill stem and answers.
3. Mark correct answers.
4. Add optional feedback.
5. Export to GIFT.

### 5. Question Types

- Single Choice
- Multiple Choice
- Matching
- True/False
- Short Answer
- Numerical
- Missing Word (cloze)

### 6. Export

Use “Export to Moodle” button. A .txt GIFT file is generated next to the .docx.

### 7. Localization

Language auto-selected by Word UI language; fallback to English.

### 8. License

Internal academic use. Adapt as needed.

## Español (es)

### 1. Propósito

Crear y exportar preguntas de cuestionarios desde Word a Moodle (formato GIFT) de forma ágil.

### 2. Características

- Botones en la cinta por tipo de pregunta.
- Inserción de plantillas.
- Ayudas de validación.
- Interfaz multilingüe.
- Soporte de pesos en respuestas.

### 3. Instalación

1. Descarga el archivo de plantilla (Word_Moodle_quiz.dotm).
2. Si Windows bloquea macros: botón derecho sobre el .dotm > Propiedades > marca “Desbloquear” > Aceptar.
3. Haz doble clic en el .dotm para crear un nuevo documento basado en la plantilla.
4. Pulsa “Habilitar contenido” para permitir macros.
5. Usa los botones de la pestaña “Exportar a Moodle”.

### 4. Flujo Básico

1. Insertar plantilla.
2. Redactar enunciado y respuestas.
3. Marcar las correctas.
4. Añadir feedback opcional.
5. Exportar a GIFT.

### 5. Tipos de Pregunta

- Elección única
- Elección múltiple
- Relación
- Verdadero/Falso
- Respuesta corta
- Numérica
- Palabra faltante

### 6. Exportación

Botón “Exportar a Moodle”. Se genera .txt GIFT junto al .docx.

### 7. Localización

Se detecta el idioma de Word; si no, inglés.

### 8. Licencia

Uso académico interno.
